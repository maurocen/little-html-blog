Create your post


You can consider this a little template you should follow, saving the file as a YAML file in the /admin/posts folder.

BE CAREFUL!

  • Don't forget to update /admin/posts/recent.yml, /admin/posts/files.yml and your latest post YAML files, as you will generate the post file but not the links for the rest of the page to link to it.
  • You may have noticed that previous and next require the html extension but filename doesn't, so it's important you don't add the extension to filename.
  • It's not really necessary to have the previous or the next camps if you're making the first or last post.
  • Don't forget the last empty line in the YAML file or the script won't be able to read it correctly, throwing an error (this took me months to notice).

This blog was made to be crazy stupid to use, so be careful, it needs to be done correctly for it to work.
